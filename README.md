# 🌦 Weather Prediction App

A Machine Learning web application built with Streamlit that predicts whether it will rain tomorrow or NOT based on weather conditions.

## Features

- Predicts rain tomorrow
- Random Forest Classifier
- Interactive Streamlit UI
- Prediction probability

## Tech Stack

- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
- Joblib

## Project Structure


weather-streamlit/
│── app.py
│── Weather_model.pkl
│── requirements.txt
│── README.md

 
## Installation


git clone https://github.com/kunal13-Ai?tab=repositories

cd weather-streamlit

pip install -r requirements.txt

streamlit run app.py


## Model
Algorithm: Random Forest Classifier
